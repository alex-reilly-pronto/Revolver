  • prepareForNewPopulation(_:)

    This method is called when the population pool changes.

    It allows the selection object to prefetch some population-specific values so that they don’t need to be recaulculated every time selection happens, ultimately accelerating the selection process.

    Warning

    This method is abstract and must be implemented in a subclass.

    Declaration

    Swift

    public func prepareForNewPopulation(population: [FitnessType])

    Parameters

    population

    Population to select from. It is guaranteed that at the time of calling this method, the fitness property of all individuals in the population is not equal to nil.
